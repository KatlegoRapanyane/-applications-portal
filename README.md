# Applications Portal 💌

A full-stack web app with a multi-step application form and hidden admin dashboard.

🔗 Live site: https://harmonious-pony-89fed7.netlify.app

## Built with
- HTML5, CSS3, JavaScript
- Firebase Firestore (database)
- Firebase Authentication (admin login)
- EmailJS (email notifications)
- Netlify deployment

## Features
- Multi-step form with validation
- Real-time data sync with Firestore
- Hidden admin panel with gesture reveal
- Application status tracking (approve/reject)
